Project 1: Weather App

Description: This program randomly generates the weather and provides a drawing based off if conditions are snowy, rainy, or sunny. The app asks for the user's guess on the day's current weather conditions, then lets you know if that guess was correct. It then will display the current weather conditions in a drawing and show you the temperature, humidity, and wind speeds.

Project 2: Duck Hunt

Description: We have chosen to create a Duck Hunt game where the user must click on ducks to capture them and score as many points as possible. The ducks will be moving around and attempting to make their capture more difficult for the player as the game progresses. This will require using functions, loops, manipulative strings, and list indexing.

Project 3: Christmas E-Card

Description: For this project, we created a Christmas-themed memory card matching game for a children's hospital. Christmas music is played throughout the game with an encouraging message at the end.

Project 4: Flappy Parrot

Description: We created a Flappy Bird-inspired game where a parrot soars through the jungle, avoiding trees. Players tap the spacebar to keep the parrot in the air while navigating obstacles.

Project 5: Fish Tank Monitor
